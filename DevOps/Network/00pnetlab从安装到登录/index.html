    PNETLab从安装到登录¶

    1.什么是PNETLab¶

    PNETlab（Packet Network Emulate Tool Lab）是一款允许下载并可以共享的社区版共享实验平台。它由两部分组成。包括PNETLab Box和PNETLab Store。

    • PNETLab Box（具有离线和在线两种模式）是一款安装在虚拟机上的运行环境。
    • PNETLab Store是一个网络平台，上面有基于PNETLab多种免费实验环境，涵盖了数据库，数据通信，操作系统等领域。

    2.硬件要求¶

    最低的笔记本电脑/PC 台式机系统要求

    先决条件：

    CPU：支持 Intel® VT-x /EPT 虚拟化的 Intel CPU 操作系统：Windows 7、8、10 或 Linux Desktop VMware Workstation 12.5 或更高版本

    VMware Player 12.5 或更高版本

    PC/笔记本电脑硬件要求
    中央处理器 Intel i5/i7（4 个逻辑处理器），在 BIOS 中启用 Intel 虚拟化
    内存 8Gb
    硬盘空间 40Gb
    网络 局域网/无线局域网
    PNETLab 虚拟机要求
    中央处理器 4/1（处理器数量/每个处理器的核心数量）启用 Intel VT-x/EPT 虚拟化引擎
    内存 6Gb或更多
    硬盘 40Gb 或更多
    网络 VMware NAT 或桥接网络适配器

    注意：最小的 PC 台式机/笔记本电脑将能够运行小型实验室。每个实验室的性能和节点数量取决于实验室中部署的节点类型。

    例子：

    IOL 基于图像的节点：每个实验室最多 40-50 个节点 Dynamips 基于图像的节点：每个实验室最多 20-25 个节点 vIOS 基于图像的节点：每个实验室最多 8-10 个节点 CSRv1000 或 XRv 基于图像的节点：每个实验室最多 2-3 个

    推荐的笔记本电脑/PC 台式机系统要求

    先决条件：

    CPU：支持 Intel® VT-x /EPT 虚拟化的 Intel CPU 操作系统：Windows 7、8、10 或 Linux Desktop VMware Workstation 12.5 或更高版本

    VW Ware Player 12.5 或更高版本

    PC/笔记本电脑硬件要求
    中央处理器 Intel i7（8 个逻辑处理器），在BIOS
    内存 32Gb
    硬盘空间 200Gb
    网络 局域网/无线局域网
    PNETLab 虚拟机要求
    中央处理器 8/1（处理器数量/每个处理器的核心数量）启用 Intel VT-x/EPT 虚拟化引擎
    内存 24Gb 或更多
    硬盘 200Gb 或更多
    网络 VMware NAT 或桥接网络适配器

    注意：PC 台式机/笔记本电脑将能够运行中小型实验室。每个实验室的性能和节点数量取决于实验室中部署的节点类型。

    例子：

    IOL 基于图像的节点：每个实验室最多 120 个节点 vIOS 基于图像的节点：每个实验室最多 20-40 个节点 基于 CSR 图像的节点：每个实验室最多 10 个

    虚拟务服器系统要求

    先决条件：

    CPU：英特尔至强 CPU，支持带有扩展页表 (EPT) 的英特尔® VT-x 操作系统：ESXi 6.0 或更高版本

    服务器硬件要求 规格
    中央处理器 推荐 CPU 2x Intel E5-2650v3（40 个逻辑处理器）或更好地支持带有扩展页表 (EPT) 的 Intel® VT-x最低 CPU 是任何支持 Intel® VT-x 的 Intel Xeon CPU带有扩展页表 (EPT)
    内存 128Gb
    硬盘空间 2Tb
    网络 局域网以太网
    服务器硬件要求 规格
    中央处理器 32/1（处理器数量/每个处理器的内核数量）启用 Intel VT-x/EPT 虚拟化引擎
    内存 64Gb 及以上
    硬盘 800Gb 及以上
    网络 vSwitch/VMnet

    注意：每个实验室的性能和节点数量取决于实验室中使用的节点类型。

    例子：

    120 IOL 基于镜像的实验室

    每个实验室 20 个 基于CSRv1000 镜像的节点

    专用服务器（裸机）系统要求

    先条件：

    CPU：Intel Xeon CPU 支持 Intel® VT-x with Extended Page Tables (EPT) 操作系统：Ubuntu Server 16.04.4 LTS x64

    服务器硬件要求 规格
    中央处理器 推荐的 CPU Intel E5-2650v3（40 个逻辑处理器）或更好的支持 Intel® VT-x with Extended Page Tables (EPT) 最低 CPU 是任何支持 Intel® VT-x 和 扩展页E表xtended Page Tables (EPT) 的 Intel Xeon CPU
    内存 128Gb
    硬盘空间 2Tb
    网络 局域网以太网

    支持的虚拟化平台和软件¶

    虚拟化平台 支持情况
    VMware Workstation 12.5 或更高版本 支持
    VMware Player 12.5 或更高版本 支持
    VMware ESXi 6.0 或更高版本 支持
    Ubuntu Server 16.04 LTS 作为裸机平台（物理机安装） 支持
    谷歌云平台 支持

    不受支持的硬件和系统¶

    硬件或者虚拟化平台 支持情况
    基于 AMD CPU 的 PC 或服务器 不支持
    VirtualBox 虚拟化 不支持
    思杰 XenServer 不支持
    微软 HyperV 不支持
    Ubuntu 17.X 或 18.x 作为平台 不支持

    3.安装PNETLab¶

    百度网盘下载链接 提取码：2333

    1.下载.ova文件，并导入VMWare Workstation。

    2.提前下载好WMware Workstation，将下载的好的PNET_4.2.10.ova文件使用VMWare Workstation文件打开。

    3.弹出以下对话框，设置好虚拟机名称和虚拟机的存储路径，点击导入按钮。

    4.等待进度条完成

    5.虚拟机导入成功后点击编辑虚拟机设置；开启该虚拟机的Intel VT-x/EPT。

    注意：虚拟机处理器要开启Intel VT-x/EPT。否则安装完成后虚拟机镜像不能正常启动。

    6.开启虚拟机，登录

    7.查看虚拟机状态是否正常

    **注意：**如果出现下图框选起来的提示信息证明已经开启了虚拟机处理器的Intel VT-x/EMP选项。如果跟下图框选出来字段不匹配，一定要开启Intel VT-x/EMP选项。

    8.使用给定的IP地址输入浏览器，登录PNETLab

    9.在浏览器点击高级选项卡，点击继续前往。

    10.出现登录界面；有两种登录模式。Online Mode（在线模式）Offine Mode（离线模式）

    11.如果选择在线登录，并且是初次登录需要创建用户名和密码，

    12.点击Sign Up，依次输入用户名，邮箱以及两次密码，验证码

    13.注册完成后返回登录界面，依据提示输入相关信息登录即可。

    14.登录成功后可以进入Store页面下载现成的网络拓扑进行练习。

    15.点击下载拓扑会弹出一个对话框，在此对话款只需要填写安装好的PNETLab提供的登录IP地址即可

    16.点击Download选项卡，页面会跳转到登录页面，输入验证码，点击Login重新登录。

    17.出现警告页面，忽略警告。点击仍然发送

    18.在此页面可以完成创建拓扑等操作，至此在线模式登录成功。

    19.离线模式和在线模式功能并无区别，除了无法访问PNETLab自带的拓扑商店，下载其他用户设计好的拓扑。如果想启用离线模式必须先处于在线模式更改系统设置。

    20.开启离线模式

    21.退出登录，页面变更为离线模式登录页面。再此页面用户名默认为admin密码为pnet默认Console选择为HTML Console。输入验证码，点击Login。即可登录离线模式。
